    Airplane Seating (Lost Boarding Pass)

    There are n passengers and n seats. Passenger 1 sits randomly; each later passenger takes their own seat if free, otherwise picks randomly. What‚Äôs the probability that the last passenger sits in their own seat? More generally, what‚Äôs the probability that the k-th passenger from the end finds their seat vacant?

    Recap

    P(A) = (number of outcomes where A happens) / (total number of outcomes). Here A is ‚Äúthe last passenger sits in the proper seat.‚Äù

    Key idea (two special seats)

    Track only seat 1 (for passenger 1) and seat n (for the last passenger). Passenger 1 chooses a seat. Anyone whose seat is taken picks a random free seat; think of a ‚Äúwrong-seat token‚Äù that jumps each time that happens. The process stops the first time the chain hits either seat 1 or seat n:

    • If the chain hits seat 1 first, seat n was never taken ‚áí the last passenger sits correctly.
    • If it hits seat n first, the last passenger‚Äôs seat is already occupied ‚áí they cannot sit correctly.

    Counting by how many sit wrong (k-view)

    Let k be the number of people who end up in the wrong seat. We never have k = 1. For each admissible k (0, 2, 3, ‚Ä¶, n) there are two symmetric ways the chain can close: at seat 1 or at seat n. Exactly half of those cases give the last passenger their correct seat.

    Result for the last passenger

    P(last sits correctly) = 1/2 for every n ‚â• 2. Symmetry between the two special seats makes them equally likely to be the first one hit.

    Generalization: k-th from last

    Consider passenger number n ‚àí k + 1 (the k-th from the end). Track the special set S = {1, n‚àík+1, n‚àík+2, ‚Ä¶, n} of size k+1. Just before that passenger boards, the wrong-seat token is equally likely to be on any seat in S. Their seat is free exactly when the token is on one of the other k seats. Hence: P(k-th from last finds own seat free) = k / (k + 1).

    Checks: k=1 ‚Üí 1/2 (the last passenger), k=2 ‚Üí 2/3, k=3 ‚Üí 3/4, independent of n (for n ‚â• k+1).

    Alternative intuition via cases

    Best case k=0: passenger 1 sits in seat 1; everyone sits correctly. For k=2 or k=3, the displacement loop either absorbs seat n (last is wrong) or closes at seat 1 (last is right). This ‚Äútwo outcomes per k‚Äù pattern again yields 1/2 overall.
