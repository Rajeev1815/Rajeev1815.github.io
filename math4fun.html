Temperature vs. Entropy in LLM Sampling

Show that increasing softmax temperature T increases output entropy for a fixed logit vector; discuss top‑p counter‑cases.

Solution
Differentiate H(p(T)) with p_i=softmax(z_i/T); dH/dT ≥ 0 under full support; truncation breaks monotonicity.

Why Soft Labels for Distillation?

Minimizing KL with softened teacher outputs preserves inter‑class structure (dark knowledge) vs one‑hot CE.

Solution
With q_T=softmax(z/T), KL(p‖q_T) transfers relative logits; one‑hot targets collapse non‑argmax information.

Airplane Seating (Lost Boarding Pass)

There are n passengers and n seats. The first passenger lost the boarding pass and picks a seat uniformly at random. Everyone else takes their own seat if free; otherwise they also pick a random free seat. What is the probability the last passenger sits in the correct seat?

Answer (last passenger)

For all n ≥ 2, the probability is 1/2.

Reasoning

  1. Track only the “special” seats: seat 1 (the first passenger’s proper seat) and seat n (the last passenger’s proper seat).
  2. Once the first passenger sits randomly, everyone with their own seat free will take it and doesn’t affect these special seats. Only when someone’s seat is taken do they choose a random free seat, which can only be one of the current “special” seats.
  3. The process ends the first time one of the two special seats is taken: either seat 1 (then the last passenger’s seat is free) or seat n (then it isn’t). By symmetry, each happens with probability 1/2.

Generalization

Let k be a positive integer with 1 ≤ k ≤ n−1. Consider the kth person from the end (passenger number n−k+1). The probability that they find their own seat vacant (hence sit correctly) is k/(k+1).

Why k/(k+1)?

  1. Extend the “special seats” set to S = { 1, n−k+1, n−k+2, …, n }, which has k+1 seats: the first passenger’s seat and the final k proper seats.
  2. Invariant: at any time before one of these special seats is finally taken, exactly one seat in S is “claimed by the wrong person” and this “wrongness token” jumps uniformly among the currently free seats in S whenever someone is forced to choose randomly. Passengers whose own seats are free just sit and don’t affect S.
  3. When passenger n−k+1 is about to board, the token is equally likely to be on any one of the k+1 seats in S. Their seat is vacant exactly when the token is on one of the other k seats. Hence P = k/(k+1).

Checks: for k=1 (the very last passenger) we get 1/2; for k=2 (second-to-last) we get 2/3; for k=3 we get 3/4, etc. The value does not depend on n as long as n ≥ k+1.

Coupon Collector Tail

Prove P(T ≥ n(ln n + c)) ≤ e^{-c} via union bound.

Solution
Bound P(coupon i unseen) and union bound over i; pick k=n(ln n + c).

GCN Over‑smoothing Threshold
